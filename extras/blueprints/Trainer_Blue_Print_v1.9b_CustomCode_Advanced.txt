  ## USAGE GUIDE (v1.9b)

  ═══════════════════════════════════════════════════════════════════════

  ### Tool Execution
  1. Navigate to Custom Code → Tools tab
  2. Enable desired tools (default: SAFE/LOW/MEDIUM enabled, CRITICAL disabled)
  3. Click "💾 Save Settings"
  4. Go to Chat tab
  5. Select and mount an Ollama model with function calling support
  6. Send messages - model will use enabled tools automatically
  7. Tool results appear in chat with ✓/✗ indicators

  ### Working Directory
  1. Go to Custom Code → Chat tab
  2. Click "📂 Change Dir" button (left of Mount)
  3. Select directory in file picker
  4. Confirms with popup and system message
  5. All tool executions use new directory

  ### Basic Settings
  1. Go to Custom Code → Settings → ⚙️ Basic
  2. Configure:
     - Working directory
     - Tool execution preferences
     - Chat behavior (auto-mount, history)
     - Project settings
     - Advanced features (debug, tool details)
  3. Click "💾 Save Custom Code Settings"

  ### Advanced Settings
  1. Go to Custom Code → Settings → 🔧 Advanced
  2. Configure 5 sections:
     - 🔄 Format Translation & Parsing
     - 🎯 Tool Intelligence
     - ⚡ Resource Management
     - 🧠 Context & RAG Optimization
     - ✅ Quality & Verification
  3. Click "💾 Save Advanced Settings"
  4. Settings persist across restarts
