#### Restore Operation (`_restore_from_backup`)
- **Location:** Lines 2555-2653
- **Event Sequence:**
  1. Verify backup and target exist
  2. Create Bug-marked copy of current file
  3. Restore backup to target location
  4. Update manifest with Bug-marked file
  5. Log operation to log.json (background)
  6. Update UI (traceback only, non-blocking)

**Key Features:**
- Bug-marked files tracked in manifest for future restoration
- File size warnings for large files (>10MB)
- Progress updates via traceback window
- Non-blocking UI refresh with `self.after(100, ...)`
- Removed all blocking `messagebox` calls

### 4. GUI Performance Fixes

#### Issue: Menu Display Blocking
- **Location:** Lines 2515-2553
- **Problem:** `tk_popup()` with `grab_release()` was blocking GUI
- **Solution:**
  - Changed to `menu.post()` with `after_idle()`
  - Added auto-dismiss on outside click
  - Proper cleanup of event bindings after 5 seconds

#### Issue: JSON I/O Blocking Main Thread
- **Locations:** Lines 2255-2292, 2294-2336
- **Problem:** Synchronous file operations freezing GUI
- **Solution:**
  - Moved to background threads using `threading.Thread(daemon=True)`
  - Added file size limits (10MB for logs, 5MB for manifest)
  - Reduced data retention for performance

#### Issue: Slow Path Resolution
- **Locations:** Lines 2355, 2412-2413, 2576-2577
- **Problem:** `Path.resolve()` slow with symlinks
- **Solution:** Changed to `Path.absolute()` (much faster)

#### Issue: Backup List Performance
- **Location:** Lines 2358-2368
- **Problem:** Checking all backup files without limits
- **Solution:**
  - Limited to checking first 50 backups max
  - Limited dropdown display to 20 most recent
  - Added try/except for each file check

## New Files Created

### 1. `log.json`
- **Location:** Same directory as grep_flight_v2.py
- **Purpose:** Logs all backup/restore operations
- **Format:**
```json
[
  {
    "timestamp": "2026-01-17T01:23:45.123456",
    "type": "backup",
    "action": "backup_created",
    "original_file": "/path/to/file.py",
    "backup_file": "/path/to/file_backup_20260117_012345.py",
    "backup_name": "file_backup_20260117_012345.py",
    "file_size": 12345
  }
]
```

### 2. `backup_manifest.json`
- **Location:** Same directory as grep_flight_v2.py
- **Purpose:** Tracks all backups by original file
- **Format:**
```json
{
  "/absolute/path/to/original.py": [
    {
      "backup_path": "/absolute/path/to/original_backup_20260117_012345.py",
      "timestamp": "2026-01-17T01:23:45.123456",
      "backup_name": "original_backup_20260117_012345.py"
    }
  ]
}
```
