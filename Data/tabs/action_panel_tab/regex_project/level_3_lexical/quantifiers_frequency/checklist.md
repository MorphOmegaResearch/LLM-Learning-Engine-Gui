# Quantifiers & Frequency Checklist

- [x] `\b(all|every|each|any)\b` (Universal Quantifiers)
- [x] `\b(some|many|few|several|most)\b` (Existential/Relative Quantifiers)
- [x] `\b(always|never|often|sometimes|rarely)\b` (Adverbs of Frequency)
- [x] `\b(none|nothing|nobody|nowhere)\b` (Negative Quantifiers)
- [x] `\b(once|twice|thrice)\b` (Multiplicative frequency)
- [x] `\b(firstly|secondly|lastly)\b` (Enumerative adverbs)
- [x] `\b(more|less|fewer)\b` (Comparative quantifiers)
- [x] `\b(enough|too many|too much)\b` (Sufficiency markers)
- [x] `\b(daily|weekly|monthly|yearly)\b` (Periodic frequency)
- [x] `\b(a lot|heaps|tons)\b` (Informal quantifiers)
