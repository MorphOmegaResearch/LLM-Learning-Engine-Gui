# Level 4 Syntax Checklist

- [x] `\b(the|a|an)\s+(\w+\s+){0,2}\w+\b` (Noun Phrase: Det + optional Adjectives + Noun)
- [x] `\b(am|is|are|was|were|have|has|had|do|does|did|can|will|should)\s+\w+ing\b` (Verb Phrase: Aux + Present Participle)
- [x] `\b(I|you|he|she|it|we|they)\s+\w+s?\s+[\w\s]+\b` (Basic SVO structure - Subject Verb Object)
- [x] `\b\w+ly\s+\w+ed\b` (Adverb-Adjective phrase)
- [x] `\b(if|because|although|when|while)\s+[\w\s,]+\b` (Subordinate Clause start)
- [x] `\bto\s+\w+\b` (Infinitive phrase)
- [x] `\b(who|which|that)\s+\w+s?\b` (Relative Clause marker)
- [x] `\b\w+,\s+\w+,\s+(and|or)\s+\w+\b` (List/Coordination syntax)
- [x] `\bis\s+not\s+\w+ing\b` (Negative Verb Phrase)
- [x] `\b\w+ed\s+by\s+\w+\b` (Passive Voice construction)
