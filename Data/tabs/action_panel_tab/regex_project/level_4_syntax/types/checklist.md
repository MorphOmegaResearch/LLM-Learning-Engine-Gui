# Syntactic Types Checklist

- [x] `\b(Do|Does|Did|Can|Could|Will|Would|Are|Is|Was|Were)\s+[\w\s]+\?` (Yes/No Question Syntax)
- [x] `\b(What|Who|Where|When|Why|How)\s+[\w\s]+\?` (WH- Question Syntax)
- [x] `^\w+\s+[\w\s]+!` (Imperative/Command Syntax - begins with Verb)
- [x] `\bDon't\s+\w+\b` (Negative Imperative)
- [x] `\b(Please|Kindly)\s+\w+\b` (Polite Imperative)
- [x] `\b(What|How)\s+a?\s+\w+!\b` (Exclamatory Syntax)
- [x] `\b[\w\s]+,\s+don't\s+you\?` (Tag Question Syntax)
- [x] `\b(Let's|Let us)\s+\w+\b` (Adhortative/Inclusive Imperative)
- [x] `\bWould\s+you\s+mind\s+\w+ing\b` (Indirect Request Syntax)
- [x] `\bIf\s+[\w\s]+,\s+then\s+[\w\s]+\b` (Conditional Syntax)
