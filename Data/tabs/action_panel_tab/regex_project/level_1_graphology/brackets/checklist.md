# Brackets & Parentheticals Checklist

- [x] `\(.*?\)` (Parentheses)
- [x] `\[.*?\]` (Square Brackets)
- [x] `\{.*?\}` (Curly Brackets/Braces)
- [x] `<.*?>` (Angle Brackets)
- [x] `\((?:[Pp]\.|[Pp]age|[Vv]erse)\s*\d+\)` (Citation parentheses)
- [x] `\(\w+(?:,\s*\d{4})?\)` (Author-Year parentheses)
- [x] `\[sic\]` (Correction marker)
- [x] `\(s\)` (Optional plural marker)
- [x] `\((?:\d{1,2}|[a-zA-Z])\)` (List item markers)
- [x] `["'].*?["']` (Nested quotation markers)
