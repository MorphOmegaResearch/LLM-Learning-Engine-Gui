# Level 1 Graphology Checklist

- [x] `[.!?]` (End of sentence punctuation)
- [x] `[,;:]` (Internal sentence punctuation)
- [x] `^[A-Z]` (Sentence start capitalization)
- [x] `\s+` (Whitespace detection)
- [x] `['"]` (Quotation detection)
- [x] `\d+` (Numeric digits)
- [x] `[A-Z]{2,}` (All-caps/Shouting detection)
- [x] `\b[A-Z][a-z]*\b` (Proper noun capitalization pattern)
- [x] `[#@&]` (Special symbol/Social tag detection)
- [x] `[-_]` (Hyphen/Underscore detection)
