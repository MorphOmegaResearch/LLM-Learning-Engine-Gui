# Level 7 Discourse Checklist

- [x] `\b(first|second|third|finally)\b` (Sequential Markers)
- [x] `\b(to begin with|first of all)\b` (Opening Transitions)
- [x] `\b(in conclusion|to sum up|overall)\b` (Closing/Summary Transitions)
- [x] `\b(by the way|speaking of|incidentally)\b` (Topic Shift Markers)
- [x] `\b(however|on the other hand|nevertheless)\b` (Contrastive Discourse)
- [x] `\b(furthermore|moreover|in addition)\b` (Additive Discourse)
- [x] `\b(as I mentioned|as previously stated)\b` (Referential/Anaphoric markers)
- [x] `\b(anyway|anyhow|moving on)\b` (Resumption/Transition markers)
- [x] `\b(for example|for instance)\b` (Exemplification markers)
- [x] `\b(consequently|as a result|therefore)\b` (Causal/Resultative Discourse)
