# Academic Domain Checklist

- [x] `\b(furthermore|moreover|nevertheless|consequently|subsequently)\b` (Academic Connectors)
- [x] `\b(hypothesize|analyze|synthesize|evaluate|verify)\b` (Scholarly Verbs)
- [x] `\b(et al\.|ibid\.|cf\.|e\.g\.|i\.e\.)\b` (Latin Citation/Abbreviation markers)
- [x] `\(\w+,\s+\d{4}\)` (Standard Citation format: (Author, Year))
- [x] `\b(significant|substantial|empirical|theoretical|methodological)\b` (Academic Adjectives)
- [x] `\b(the data suggests|the evidence indicates|it is argued that)\b` (Hedged Academic Assertions)
- [x] `\b(nominalization|standardization|utilization|classification)\b` (Suffix -ation/ization - Formal Nouns)
- [x] `\b(it is observed that|it has been noted that)\b` (Passive voice/Impersonal construction)
- [x] `\b(the results of|the analysis of|the study of)\b` (Genitive Phrasal markers)
- [x] `\b(with respect to|in terms of|as regards)\b` (Formal prepositional phrases)
