# Intent Domains Checklist

- [x] `\b(I am informing you|I'd like to point out|did you know)\b` (Informative: Stating facts)
- [x] `\b(I command|you must|I require|you are ordered)\b` (Directive: Orders/Commands)
- [x] `\b(I promise|I will ensure|I guarantee|I commit)\b` (Commissive: Promises/Commitments)
- [x] `\b(I apologize|I'm sorry|pardon me)\b` (Expressive: Social/Emotional)
- [x] `\b(I declare|I pronounce|this is now)\b` (Declarative: Status change)
- [x] `\b(I suggest|I recommend|you might want to)\b` (Directive: Suggestion/Advice)
- [x] `\b(I refuse|I cannot|I decline|no way)\b` (Commissive: Refusal)
- [x] `\b(I wonder|I ask|can you tell me)\b` (Informative: Inquiry/Questioning)
- [x] `\b(I agree|I concur|exactly|you're right)\b` (Expressive: Agreement)
- [x] `\b(I object|I disagree|that's not true)\b` (Expressive: Disagreement)
