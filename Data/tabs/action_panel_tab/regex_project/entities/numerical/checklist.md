# Numerical Entities Checklist

- [x] `\b\d+\b` (Integers)
- [x] `\b\d+\.\d+\b` (Decimals)
- [x] `\b(\$|£|€|¥)\s?\d+(?:\.\d{2})?\b` (Currency)
- [x] `\b\d+\s?%\b` (Percentages)
- [x] `\b\d+\s?(kg|g|mg|lbs|oz)\b` (Weight measurements)
- [x] `\b\d+\s?(m|cm|mm|km|in|ft|yd|mi)\b` (Distance/Length)
- [x] `\b\d+\s?(l|ml|gal|qt|pt)\b` (Volume)
- [x] `\b\d+\s?(°C|°F|K)\b` (Temperature)
- [x] `\b(one|two|three|four|five|six|seven|eight|nine|ten)\b` (Cardinal number words)
- [x] `\b(first|second|third|fourth|fifth)\b` (Ordinal number words)
