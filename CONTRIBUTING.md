# Contributing to LLM-Learning-Engine-Gui

Development is coordinated — please open an issue or contact via GitHub
before submitting pull requests.

## How to engage

- **Questions / discussion** — open a GitHub issue
- **Research collaboration** — contact via [MorphOmegaResearch](https://github.com/MorphOmegaResearch)
- **Commercial licensing** — contact via GitHub before any commercial use
